``` ### Cara Menjalankan 1. **Siapkan skrip contoh** seperti `script_satu.py` dan `script_dua.py`. Pastikan mereka menggunakan modul `logging` untuk menulis ke file `.log` yang sesuai. 2. **Jalankan Skrip Anda:** Buka beberapa terminal dan jalankan skrip Python yang ingin Anda pantau. ```bash # Di terminal 1 python script_satu.py # Di terminal 2 python script_dua.py ``` 3. **Jalankan Dasbor:** Buka terminal lain, pastikan Anda berada di folder `dasbor_monitor`, dan jalankan server Flask. ```bash python app.py ``` 4. **Buka di Browser:** Buka browser web Anda dan akses alamat `http://127.0.0.1:5000`.
